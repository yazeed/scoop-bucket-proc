[Scoop](https://scoop.sh) bucket for [proc](https://github.com/yazeed/proc).

## Install

```powershell
scoop bucket add proc https://github.com/yazeed/scoop-bucket-proc
scoop install proc
```

## Update

```powershell
scoop update proc
```
